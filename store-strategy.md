## 22. Mobile-First UI/UX Structure

### Breakpoints
| Device | Width | Notes |
|--------|-------|-------|
| Phone | < 480px | Single column, stacked sections |
| Tablet | 481-768px | 2 column grids |
| Desktop | 769-1024px | Full layout |
| Wide | > 1024px | Max width 1200px |

### Mobile Specifics
- **Hero**: Full screen, centered text, CTA buttons stacked
- **Product**: Image first, details below, sticky ATC on scroll
- **Benefits**: Single column cards
- **Testimonials**: Single column, swipeable on touch
- **Grids**: 2 columns on mobile (product grid)
- **Touch targets**: 44px minimum tap targets
- **Font sizes**: No smaller than 14px on mobile
- **Spacing**: 16px gutters, 40px section padding

### Performance
- Compressed images (WebP where supported)
- Lazy loading below-fold images
- Font subsetting (Latin + Arabic only needed chars)
- CSS minified
- JS deferred
- CDN for assets
